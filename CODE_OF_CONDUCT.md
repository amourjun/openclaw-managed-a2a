# Code of Conduct

## Our Standard

This project is built for rigorous technical collaboration. Participants are expected to:

- communicate clearly and respectfully
- focus discussion on evidence, behavior, and trade-offs
- challenge ideas without attacking people
- keep feedback actionable and scoped to the work

## Unacceptable Behavior

The following are not acceptable in project spaces:

- harassment, discrimination, or personal attacks
- intimidation or abusive language
- bad-faith derailment, spam, or repeated low-signal disruption
- sharing private or security-sensitive information without permission

## Enforcement

Project maintainers may edit, remove, or reject comments, issues, or pull requests that violate this code of conduct.

Repeated or serious violations may result in temporary or permanent removal from project spaces.

## Reporting

If you experience or witness unacceptable behavior, contact the maintainer through GitHub: `@amourjun`.
